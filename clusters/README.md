# Example Clusters

This directory shows a simple GitOps-oriented environment layout for `dev` and `prod`.

The examples are intentionally small:

- `dev/` composes a lower-risk environment baseline
- `prod/` composes a production-style baseline

Use these examples as a starting point for:

- cluster directory structure
- environment-specific composition with Kustomize
- CI validation of rendered Kubernetes manifests

These examples are illustrative and should be adapted before production use.
