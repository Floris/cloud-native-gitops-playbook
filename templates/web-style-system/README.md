# Web Style System Template

This template captures the baseline design tokens for Floris-managed websites.

Use it when:

- creating a new site in the portfolio ecosystem
- restyling an existing site toward the shared house style
- asking an AI system to build UI that matches `json.droppert.dev`

## Contents

- `base/tokens.css`: shared CSS custom properties and foundational styling hooks

## Usage

Import the token file early in the page or asset pipeline, then layer site-specific components on top.

Do not treat this template as a complete component library. It is the common visual foundation.
